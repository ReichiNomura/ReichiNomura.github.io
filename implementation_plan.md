# オセロゲーム実装計画書

## 目標
現在のリポジトリのサブディレクトリ（`/othello`）に、Webブラウザで動作するオセロ（リバーシ）ゲームを作成します。

## 実装済み機能
- **ゲームモード**:
    - 「人間 vs CPU」
    - 「CPU vs CPU」
- **手番選択**:
    - 「人間 vs CPU」の場合、先攻（黒）か後攻（白）を選択可能。
- **CPUロジック**:
    - 角（隅）を優先的に確保する戦略。
    - それ以外はランダムに有効手を選択。
- **言語対応**:
    - UIメッセージ、コンソールログ、コード内コメントはすべて日本語。

## ディレクトリ構成
`othello/`

### `othello/index.html`
- メインのゲームインターフェース。
- 指定された対戦モードと手番選択のUI。

### `othello/style.css`
- ゲームのスタイル設定。
- 緑の盤面と黒・白の石のデザイン。

### `othello/game.js`
- ゲーム進行制御クラス (`OthelloGame`)。
- 盤面管理、有効手判定、石の反転、ターン管理、終了判定。

### `othello/cpu.js`
- CPUロジッククラス (`CPU`)。
- `decideMove`: 盤面を評価して次の一手を決定。

## 検証項目
### 手動検証
- [x] **設定確認**:
    - 「人間 vs CPU」で先攻（黒）を選び、自分が初手であることを確認。
    - 「人間 vs CPU」で後攻（白）を選び、CPUが初手を打つことを確認。
    - 「CPU vs CPU」を選び、自動でゲームが進行し、終局まで進むことを確認。
- [x] **ゲームプレイ**:
    - 通常のオセロルール通りに石が返るか。
    - パスが発生する場合、適切に処理されるか（"スキップ"表示）。
    - 終局時に正しいスコアと勝者が表示されるか。
